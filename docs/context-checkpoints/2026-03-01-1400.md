### 2. Bug: Missing enum values ✅
**Root cause:** S40 migration's `ALTER TYPE ... ADD VALUE` for `staff`, `site_manager`, `auditor` never committed to the DB (silent transaction failure in Supabase SQL editor).
**Fix:** Ran manually in SQL editor:
```sql

ALTER TYPE public.user_role ADD VALUE IF NOT EXISTS 'site_manager';
ALTER TYPE public.user_role ADD VALUE IF NOT EXISTS 'auditor';
```

**Fix:** Changed to `.neq('is_active', false)` — includes null rows.
